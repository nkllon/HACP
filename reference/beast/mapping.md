# Aegis ↔ HACP Mapping (reference)

## Purpose

Map Aegis engineering patterns to HACP artifacts to facilitate conversations and pilots.

- Open interfaces → Task Manifest & Role Declarations
- Rules-of-Engagement → Authority Principle
- Verification / Certification → HACP compliance levels (HACP-2)
- Escalation / Abort → Escalation & Refusal rules
- After-Action Review → Decision path & Auditability

For full brief and pilot plan see `docs/pilot/sensor-fusion-pilot.md`.
